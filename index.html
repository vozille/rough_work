Bond and Addition

ProblemsMy SubmissionsAll SubmissionsSubmit


Author:
Anwesh Mohanty

Statement :

Bond is a famous teacher in CET. CET is weird in the sense that students of one branch only study one subject. One of the favorite hobbies of Bond include making a list of numbers secured by students of a branch. As his name suggests, he likes to bond things together. So he joins all the numbers end to end. He does so for multiple branches. Finally he adds all such numbers and keeps the result. He gets a lot of satisfaction by looking at the resulting number.

However since Bond is so busy , he is unable to indulge in his favorite hobby. Your task is to get the final result from the data bond has given you. Success would mean some grace marks in internals , so there's a lot at stake.

Are you up to the mark for this tremendous undertaking ?

Input:
First line contains an integer T, the number of testcases.
Next line contains an integer N, the number of branches.
Next N lines contain the following pattern:
An integer K denoting the number of students in the branch
K integers denoting the marks of each student.

Output:
Print the Answer in a new line for each testcase

Constraints:



  • 1 <= T <= 100

  • 1 <= N <= 1000

  • 1 <= K <= 1000

  • 0 <= marks <= 100


Sample Input:
1
3
5
10 20 30 40 50
1
100
2
1 1

Sample Output:
1020304161

Explanation:
There are 3 Branches. First branch contains 3 students, Second has only 1 and 3rd has 2.
On joining the marks from end to end, for the first branch, we get the number 1020304050
Similarly we get 100 and 11.
On adding these 3 numbers, we get the required answer.


Time Limit : 1 Second(s)
Score : 50 Point(s)
Input File Limit : 50000 Bytes
Languages Allowed : AWK,Bash,Brain,C,C++,Java,C#,JavaScript,Pascal,Perl,PHP,Python,Python3,Ruby,Text
